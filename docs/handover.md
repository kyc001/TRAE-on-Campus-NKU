# 项目交接文档

## 项目概述
- 项目名称：TRAE-on-Campus-NKU
- 项目类型：Vibe Coding Hackathon项目
- 项目目标：开发一个基于大模型的「期末复习知识网络生成工具」，帮助学生将PPT/PDF课件快速压缩成可展开的复习框架地图

## 技术方案
- **前端**：React + TypeScript + Vite + 自定义树形组件
- **后端**：Node.js + Express + TypeScript
- **API集成**：预留了大模型API接口（deepseek或豆包）
- **构建工具**：Vite

## 核心功能实现

### 后端服务
- **文件上传接口**：支持PPT/PDF文件上传
- **文档处理服务**：调用文档理解API解析文档内容
- **大模型调用服务**：将解析内容发送给大模型，生成结构化JSON
- **结果缓存机制**：处理API超时和失败情况
- **任务状态管理**：支持异步任务处理和状态查询

### 前端应用
- **文件上传组件**：支持点击和拖拽上传
- **文本输入功能**：作为文件上传的降级方案
- **知识网络可视化**：树形结构展示，支持展开/折叠节点
- **加载状态管理**：提供友好的加载提示
- **错误处理**：显示清晰的错误信息
